Introduction


Face identification is the most popular technique to use the attendance marking and person tracking with the gender, age details. Face recognition task of identifying the faces like already known and unknown or new person faces. And identify the gender, age and emotion result using face recognition process.

Using Mongodb database to store the attendance data's with details. The face recognition is based on two different process or categories. First one is Feature encoding and comparison process, and Second one is image encoding and comparison process. In this project, I am used the image encoding and comparison process for face detection and identification.

Project Working Concepts:


In this project, using face recognition process to detect the face and identify the person name, age, gender, emotion using face recognition process and custom CNN model for gender detection and age detection process. And make attendance of detected persons with date, time, gender, etc details. Coming to face recognition process, first process is prepare the required datasets, and calculate the number of classes with class names. And then , second process is data preprocessing . Data preprocessing like data encoding process. Data preprocessing for face detection using images is using two steps, one is face locations using required data's (images), and second is face encoding process. Face encoding is work like , analyze and store the feature of given face and convert to numerical representation of those faces.

Second process is connect the required camera and collect the video frames. After collected the locate the face and encode the frame using face locations and face encoding methods. and compare the trained encoding values and given frame values. After finding the match, find the match index name or class of this given matched value. Display the given class name using opencv python library tools. Coming to the gender, and age detection process, collect the separate database for those two models, and preprocess the all data's like image to array conversion, image data generator. After image data generator process, calculate the number of classes, and class names.

Create the custom CNN model for detect the gender, age with training image data's. Custom CNN model creation is carry three layers, Convolutional layer, Flatten layer, Fully connected layer. Convolutional layer is the input layer of CNN model, Flatten layer is working like, make hidden layers are flatten of this model. Connect the all input, hidden, and output layers are working in Fully connected layer.

Detect the gender, age, emotion using given camera frame. Preprocess the given frame and compare the preprocessed frame data and trained data's using model prediction process. Convert the model prediction value into array, and normalization value. Compare the normalization value to given labels. Finally get output of the gender, age, emotion details.

After detect the face with gender, date, detection time details, automatically add those details in the given MongoDB collections. Also automatically send the attendance details to required person mail. Once add the details in the given collection, avoid the duplications data's without hold or stop the detection process.
